# Agentic Autonomous Data Analyst

An autonomous, agent-based data analysis system that performs:
- Exploratory Data Analysis (EDA)
- Trend detection
- Anomaly detection
- Automated visualizations
- Insight & recommendation generation

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- llama-cpp (local LLM)
- Google Colab

## Architecture
- Planner Agent (rule-based)
- Execution Agent (safe Python execution)
- Insight Agent (LLM-powered summarization)

## How to Run
1. Upload a CSV dataset
2. Run `run_agent(question, df)`
3. Get trends, anomalies, and recommendations
